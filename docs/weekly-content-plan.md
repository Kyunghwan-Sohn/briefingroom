# 주간 브리핑 콘텐츠 기획서

## 1. 현황 분석 (Plan)

### 보유 데이터
- 11일치 1,275건 보도자료 (일평균 ~120건)
- 요약 완료율: 96.3% (1,208/1,275)
- 키워드 추출 완료
- 5개 분야 분류 완료 (금융경제/사회복지/산업기술/외교안보/행정법제)

### 현재 텔레그램 채널 운영
- 일일 브리핑 1~2회 (오전/오후)
- 분야별 핵심 3건 + 관련 뉴스 포함
- 메시지 길이 ~4,000자

### 문제 인식
- 일일 브리핑만으로는 정책 흐름/트렌드 파악 불가
- 독자가 매일 보지 못하면 맥락을 놓침
- 키워드 트렌드, 부처별 활동량 등 인사이트 미활용
