## Executive Summary

BriefingRoom은 B2B SaaS 제품(금융사 준법감시팀 대상 규제 모니터링)이지만, 인스타그램을 **브랜드 인지도 확보 + 콘텐츠 권위 구축 + 간접 리드 확보** 채널로 활용한다. 핵심 전략은 "정책/규제 정보를 시각적으로 쉽게 전달하는 정보 계정"으로 포지셔닝하여, 금융/정책 관심층을 넓게 확보한 뒤 B2B 리드로 전환하는 간접 퍼널이다.

1인 운영 제약을 감안하여, Canva Pro의 브랜드킷/템플릿/예약 발행 기능을 최대한 활용한 주 3-4회 발행 체제를 설계하며, 월 총 투입 시간을 8-10시간 이내로 관리한다.

| **슬로건** | "51개 부처 보도자료, AI가 요약합니다" |

| **시각적 아이덴티티** | 네이비(#1B2838) + 화이트 + 액센트 블루(#3B82F6) 기반 미니멀 디자인 |

## 1.2 B2B 서비스가 인스타그램을 쓰는 이유와 전략

### 왜 인스타그램인가?

B2B 서비스의 인스타그램 활용은 **직접 리드 확보가 아닌, 간접적 브랜드 구축** 관점에서 접근해야 한다.

| 전략적 근거 | 설명 |
|------------|------|
| **정보 콘텐츠 검색 엔진화** | 2026년 인스타그램은 검색 엔진 역할을 수행. "금융 규제", "정부 정책" 검색 시 노출되는 정보 계정이 될 수 있음 |
| **카드뉴스 포맷의 높은 적합성** | 보도자료 요약 = 카드뉴스와 완벽한 궁합. 캐러셀 포맷이 복잡한 정보를 슬라이드로 분해하는 데 최적 |
| **의사결정자의 개인 시간 침투** | 준법감시팀 담당자도 퇴근 후 인스타를 봄. 업무 외 시간에 브랜드 인지도를 쌓는 간접 접점 |
| **크로스 채널 콘텐츠 재활용** | 웹사이트/텔레그램/뉴스레터 콘텐츠를 시각화하여 추가 채널로 확장. 신규 콘텐츠 제작 부담 최소화 |
| **조직 내 바이럴** | "이 계정 유용해요"라는 동료 추천이 B2B 의사결정에 간접 영향 |

### 타겟 오디언스 (3-Layer)

```
[1차 타겟: 넓은 그물]
 정책/규제에 관심 있는 2030 직장인
 (공무원 준비생, 사회 초년생, 금융권 주니어)
 → 팔로워 볼륨 확보, 도달 극대화

  [2차 타겟: 전문가층]
   금융/법률/GR 업계 실무자
   (컴플라이언스 담당자, 변호사, GR 컨설턴트)
   → 콘텐츠 권위 구축, 전문 네트워크 형성

    [3차 타겟: 의사결정자]
     B2B 구매 결정권자
     (준법감시팀장, CISO, 법무팀장)
     → 간접 인지 → 링크드인/웹사이트로 유도 → 리드 전환
```

# 2. 콘텐츠 전략

## 2.1 콘텐츠 필러 (Content Pillars) 정의

5개 콘텐츠 필러를 정의하고, 각 필러에 맞는 포맷과 발행 빈도를 설정한다.

### Pillar 1: 오늘의 정책 브리핑 (핵심 콘텐츠 - 40%)

| 항목 | 내용 |
|------|------|
| **목적** | 매일 발표되는 주요 정책/보도자료를 시각적으로 요약 전달 |
| **포맷** | 캐러셀 카드뉴스 (4-6장) |
| **빈도** | 주 3회 (월/수/금) |
| **콘텐츠 구조** | 표지(제목+부처) → 핵심 내용 1 → 핵심 내용 2 → 영향/의미 → CTA |
| **소스** | 당일 BriefingRoom 웹사이트 AI 요약 콘텐츠 재가공 |
| **제작 시간** | 건당 15-20분 (템플릿 활용) |
| **예시 주제** | "금융위, 가상자산 이용자 보호법 시행령 개정안 발표 - 핵심 3가지" |

### Pillar 2: 주간 정책 TOP 5 (시그니처 콘텐츠 - 20%)

| 항목 | 내용 |
|------|------|
| **목적** | 한 주의 가장 중요한 정책 5개를 큐레이션하여 "이것만 보면 된다" 가치 제공 |
| **포맷** | 캐러셀 카드뉴스 (7-8장) 또는 릴스 (60초) |
| **빈도** | 주 1회 (일요일 저녁 또는 월요일 아침) |
| **콘텐츠 구조** | 표지 → #5 → #4 → #3 → #2 → #1 → 종합 코멘트 → CTA |
| **소스** | 주간 리포트(weekly.py) 데이터 기반 |
| **제작 시간** | 건당 30-40분 |
| **예시 주제** | "이번 주 정책 TOP 5: 1위 - 디지털 플랫폼 규제법 국회 통과" |

### Pillar 3: 규제/정책 해설 (교육 콘텐츠 - 20%)

| 항목 | 내용 |
|------|------|
| **목적** | 복잡한 규제/정책을 쉽게 풀어서 설명, 전문성과 권위 구축 |
| **포맷** | 캐러셀 카드뉴스 (6-10장) + 텍스트 중심 인포그래픽 |
| **빈도** | 주 1회 (화 또는 목) |
| **콘텐츠 구조** | 표지(궁금증 유발) → 배경 설명 → 핵심 포인트 → 영향 분석 → 타임라인 → CTA |
| **제작 시간** | 건당 40-60분 (리서치 포함) |
| **예시 주제** | "금융소비자보호법, 도대체 뭐가 바뀌나? (5분 정리)", "ESG 공시 의무화 - 우리 회사도 해당될까?" |

### Pillar 4: 서비스/비하인드 (브랜드 콘텐츠 - 10%)

| 항목 | 내용 |
|------|------|
| **목적** | BriefingRoom 서비스 소개, 개발 비하인드, 1인 개발자 스토리로 친밀감 형성 |
| **포맷** | 릴스 (15-30초) + 단일 이미지 + 스토리 |
| **빈도** | 격주 1회 |
| **콘텐츠 구조** | 가벼운 톤, 개발자의 일상, AI 요약 과정 보여주기, 서비스 업데이트 공지 |
| **제작 시간** | 건당 15-20분 |
| **예시 주제** | "1인 개발자가 51개 부처 보도자료를 매일 수집하는 방법", "AI가 보도자료를 요약하는 과정 (30초)" |

### Pillar 5: 참여 유도 (인게이지먼트 콘텐츠 - 10%)

| 항목 | 내용 |
|------|------|
| **목적** | 팔로워 참여를 유도하여 알고리즘 도달률 향상 |
| **포맷** | 스토리 투표/퀴즈 + 단일 이미지 |
| **빈도** | 주 1-2회 (스토리) |
| **콘텐츠 구조** | 정책 관련 OX 퀴즈, 투표 ("이 정책 찬성? 반대?"), 이번 주 가장 관심 가는 정책 투표 |
| **제작 시간** | 건당 5-10분 |
| **예시 주제** | "금융위 발표 - 가상자산 규제, 찬성 vs 반대?", "퀴즈: 한국 정부 부처는 몇 개일까?" |

## 2.3 Canva Pro 활용 콘텐츠 제작 효율화 방안

1인 운영의 핵심은 **템플릿 기반 배치 제작 + 예약 발행**으로 주 1회 집중 작업(2-3시간)으로 한 주 콘텐츠를 완성하는 것이다.

### Step 1: 브랜드킷 설정 (초기 1회, 30분)

| 브랜드킷 항목 | 설정값 |
|--------------|--------|
| **Primary Color** | #1B2838 (네이비) |
| **Secondary Color** | #3B82F6 (블루) |
| **Accent Color** | #F59E0B (앰버 - 강조용) |
| **Background** | #FFFFFF (화이트) / #F8FAFC (라이트 그레이) |
| **Primary Font** | Pretendard Bold (제목) |
| **Secondary Font** | Pretendard Regular (본문) |
| **로고** | BriefingRoom 로고 (수평/정방/아이콘 3종 업로드) |

### Step 2: 마스터 템플릿 제작 (초기 1회, 2-3시간)

필러별 마스터 템플릿을 만들어 매주 내용만 교체한다.

| 템플릿 | 장수 | 구성 | 재사용 방식 |
|--------|------|------|------------|
| **정책 브리핑 템플릿** | 6장 세트 | 표지/본문x3/영향/CTA | 제목+본문 텍스트만 교체 |
| **주간 TOP 5 템플릿** | 8장 세트 | 표지/5-4-3-2-1위/코멘트/CTA | 순위별 내용 교체 |
| **정책 해설 템플릿** | 10장 세트 | 표지/배경/포인트x4/타임라인/요약/CTA | 주제별 내용 교체 |
| **비하인드 템플릿** | 단일 이미지 | 사진+텍스트 오버레이 | 사진+캡션 교체 |
| **스토리 퀴즈 템플릿** | 3장 세트 | 질문/힌트/정답 | 질문 내용 교체 |

### Step 3: 주간 배치 제작 워크플로 (매주 일요일 또는 토요일, 2-3시간)

```
[1단계] 소재 수집 (30분)
  - BriefingRoom 웹사이트에서 한 주간 주요 보도자료 확인
  - 주간 리포트(weekly.py) 결과 확인
  - 다음 주 콘텐츠 주제 5개 선정

[2단계] Canva 배치 제작 (90분)
  - 마스터 템플릿 복제 → 내용 교체
  - 정책 브리핑 3건 (45분) = 건당 15분
  - 주간 TOP 5 1건 (20분)
  - 해설 콘텐츠 1건 (25분, 리서치 별도)

[3단계] 예약 발행 (15분)
  - Canva Pro Content Planner에서 일괄 예약
  - 월/화/수/금/일 시간대별 예약 설정

[4단계] 스토리/비하인드 (수시, 주 15분)
  - 스토리 퀴즈: 당일 즉석 제작 (5분)
  - 비하인드: 격주 1건, 간단 촬영+편집 (10분)
```

## 3.2 해시태그 전략

### 2026년 해시태그 정책 변화 반영

인스타그램이 2026년부터 게시물당 해시태그를 **최대 5개로 제한**했다. 따라서 소수 정밀 해시태그 전략이 필수다.

2. #보도자료요약 (니치)
3. #[해당부처명] 예: #금융위원회 (구체적)

## 3.4 B2B 리드 전환 퍼널

인스타그램에서 직접 B2B 리드를 전환하기보다, **인지 → 관심 → 신뢰 → 전환** 단계를 거치는 간접 퍼널을 설계한다.

### 퍼널 구조

```
[인스타그램]                    [전환 채널]

TOFU (인지)
 정책 카드뉴스로
 브랜드 인지도 확보
        │
        ▼
MOFU (관심)
 정책 해설로 전문성 인식
 프로필 방문 → 링크 클릭        → govbrief.kr 방문
        │                           │
        ▼                           ▼
MOFU (신뢰)                     뉴스레터 구독 (스티비)
 지속적 팔로우 +                 텔레그램 채널 가입
 스토리 인터랙션                      │
        │                           ▼
        ▼                      주간 리포트 수신
BOFU (전환)                     (무료 → 유료 전환 유도)

                                    │

```

### 퍼널 단계별 인스타 콘텐츠 역할

| 퍼널 단계 | 인스타 콘텐츠 | KPI | 전환 목표 |
|----------|-------------|-----|----------|
| **TOFU** | 정책 브리핑 카드뉴스, 릴스 | 도달(Reach), 노출(Impression) | 프로필 방문 |
| **MOFU** | 정책 해설, 주간 TOP 5 | 프로필 방문, 외부 링크 클릭 | 웹사이트 방문, 뉴스레터 구독 |
| **BOFU** | Pro 서비스 소개, 후기/사례 | 링크 클릭, DM 문의 | 무료 체험 신청 |

|------|-------------|----------|
| 팔로워 | 800-1,200 | 주 4-5 게시물, 해시태그 + 크로스 채널 |
| 월 프로필 방문 | 1,500-3,000 | 팔로워 대비 2-3배 |

**핵심 인사이트**: 인스타그램 단독으로 B2B 리드를 직접 전환하기보다, **브랜드 인지도 + 콘텐츠 자산 축적** 채널로 활용하고, 실제 전환은 뉴스레터/링크드인/직접 아웃리치에서 이루어지도록 설계한다.

# 5. 리스크 및 고려사항

## 5.1 B2B 서비스의 인스타그램 한계

| 한계 | 상세 | 완화 방안 |
|------|------|----------|
| **직접 전환의 어려움** | 준법감시팀장이 인스타에서 B2B SaaS를 결제하지 않음. 인스타그램에서의 직접 리드 전환 기대치를 낮게 설정해야 함 | 인스타를 TOFU/MOFU 채널로만 활용. 전환은 링크드인/이메일/직접 아웃리치에 집중 |
| **타겟 미스매치** | 인스타 주 사용자층(2030)과 B2B 의사결정자(40-50대 팀장급)의 괴리 | 2030 실무자를 통한 bottom-up 인지도 전략: 실무자가 팀장에게 "이런 서비스 있어요" 추천 |
| **ROI 측정 난이도** | 인스타그램 → B2B 매출 기여를 직접 추적하기 어려움 | UTM 태깅 철저히 적용. 인스타를 "브랜드 자산" 관점으로 평가 (직접 ROI가 아닌 간접 기여) |
| **콘텐츠 깊이 한계** | 인스타그램은 깊이 있는 전문 콘텐츠에 적합하지 않음 | 인스타에서는 "미끼"(후킹)만, 상세 내용은 웹/뉴스레터로 유도 |
| **알고리즘 변동** | 인스타그램 알고리즘이 수시로 변경되어 도달률 예측 불가 | 팔로워 기반이 아닌 콘텐츠 품질 기반 전략. 채널 다각화로 리스크 분산 |

## 5.2 1인 운영 시 리소스 배분

### 주간 시간 배분 예산: 총 3시간

| 활동 | 시간 | 비중 |
|------|------|------|
| 소재 수집 + 주제 선정 | 30분 | 17% |
| Canva 배치 콘텐츠 제작 | 90분 | 50% |
| 예약 발행 + 캡션 작성 | 20분 | 11% |
| 댓글/DM 응대 + 소통 | 20분 | 11% |
| 스토리 제작 (수시) | 15분 | 8% |
| 인사이트 체크 | 5분 | 3% |

### 번아웃 방지 원칙

| 원칙 | 설명 |
|------|------|
| **배치 제작 고수** | 매일 콘텐츠를 만들지 않는다. 주 1회 집중 작업으로 한 주치를 완성 |
| **완벽주의 금지** | 80% 퀄리티로 발행. 정보 정확성만 확인하고, 디자인 완벽은 추구하지 않음 |
| **자동화 우선** | Canva 예약 발행으로 매일 접속 필요 제거. CSV Bulk Create로 반복 작업 최소화 |
| **콘텐츠 재활용** | 신규 콘텐츠 비중 30%, 기존 콘텐츠(웹/텔레그램) 재가공 70% |
| **측정 최소화** | 주 1회 15분 인사이트 체크로 충분. 매일 수치 확인하지 않음 |
| **중단 기준 설정** | 3개월 후 월 외부 링크 클릭 50건 미만이면, 인스타 투자 축소 또는 중단 검토 |

### 인스타그램이 다른 채널 운영에 미치는 영향

| 기존 채널 | 영향 | 대응 |
|----------|------|------|
| 텔레그램 | 거의 없음 (자동화 운영) | 인스타 교차 홍보 문구만 추가 |
| 뉴스레터 | 소재 재활용으로 오히려 시너지 | 인스타 카드뉴스 → 뉴스레터 이미지 활용 |
| 링크드인 | 인스타 카드뉴스 → 텍스트 변환으로 효율화 | 인스타 콘텐츠가 링크드인 소재가 됨 |
| 웹사이트 | 영향 없음 (자동 운영) | 인스타 위젯 추가만 고려 |
| **B2B 영업** | 주의 필요: 인스타에 시간 과투자하면 영업 시간 줄어듦 | 주 3시간 상한선 엄수 |

## 5.3 대안 채널 비교 (ROI 관점)

### 채널별 ROI 비교

| 채널 | B2B 리드 직접 전환 | 브랜드 인지도 | 운영 시간/주 | 1인 운영 적합도 | 종합 ROI |
|------|-------------------|-------------|-------------|---------------|---------|
| **링크드인** | 매우 높음 | 높음 | 3-4시간 | 높음 | 최상 |
| **뉴스레터 (스티비)** | 높음 | 중간 | 2-3시간 | 높음 | 상 |
| **텔레그램** | 중간 | 중간 | 1시간 (자동화) | 매우 높음 | 상 |
| **인스타그램** | 낮음 | 높음 | 2.5-3시간 | 중간 | 중 |
| **유튜브** | 중간 | 매우 높음 | 8-10시간+ | 낮음 | 하 |
| **블로그/SEO** | 높음 (장기) | 높음 | 3-5시간 | 높음 | 상 (장기) |
| **X (트위터)** | 낮음 | 중간 | 1-2시간 | 높음 | 중하 |

### 채널 우선순위 권고

```
[우선순위 1 - 필수 유지]
  링크드인: B2B 리드의 핵심 채널. 인스타보다 직접 전환 효과 높음
  뉴스레터: 이메일 리스트는 소유 가능한 자산. 리드 너처링 핵심
  텔레그램: 자동화 운영으로 시간 투자 최소. 기존 사용자 유지

[우선순위 2 - 보조 채널 (인스타그램의 위치)]
  인스타그램: 브랜드 인지도 + 콘텐츠 자산 축적. 직접 전환 기대는 낮추되,
             콘텐츠 재활용 효율이 높아 추가 채널로 가치 있음

[우선순위 3 - 추후 고려]
  블로그/SEO: 장기 유기 트래픽 확보. 시간이 나면 시작
  유튜브: 1인 운영에 너무 시간 소모. 현 단계에서는 부적합
```

### 인스타그램 투자 판단 기준

- 주간 운영 시간 3시간 이하 유지
- 콘텐츠 재활용으로 다른 채널(링크드인/뉴스레터)에도 시너지 발생

**축소/중단 조건**:
