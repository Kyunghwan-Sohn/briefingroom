# 정부 지원사업 통합 모니터링 서비스 기획

> 작성일: 2026-03-31
> govbrief.kr의 보도자료 서비스와 별개로, 정부 지원사업/보조금/공모 정보를 통합 수집하여 제공

---

## 1. 서비스 개요

### 왜 필요한가

| 현재 문제 | 구체적 상황 |
|----------|----------|
| **분산** | 지원사업이 기업마당, 중소벤처24, K-Startup, NTIS 등 9개 이상 사이트에 흩어져 있음 |
| **발견 지연** | 공고 후 2~3일 뒤에 알게 되어 준비 시간 부족 |
| **자격 판단 어려움** | 공고문이 30~50페이지 PDF, 핵심 자격 요건 파악에 30분+ 소요 |
| **마감 놓침** | 여러 사이트를 매일 확인할 여력 없어 적격 사업 놓침 |
| **유료 서비스 비쌈** | 기존 지원사업 알림 서비스 월 30~100만원 |

### 서비스 컨셉

```
정부 지원사업 통합 크롤링 → AI 요약 (자격/금액/기간) → 텔레그램 알림 + 웹사이트
```

**한 줄 정의**: "모든 정부 지원사업을 AI가 매일 모아서, 핵심만 요약해서, 텔레그램으로 알려드립니다."

## 2. 데이터 소스 전략

### 2.1 API 기반 (1순위 — 안정적, 구조화)

| 소스 | API | 커버리지 | 일일 한도 | 데이터 |
|------|-----|---------|---------|--------|
| **기업마당** (bizinfo.go.kr) | REST (JSON/XML) | 중앙부처+지자체+유관기관 통합 | 개발 1,000건/일 | 사업명, 신청기간, 소관기관, 분야, 상세URL |
| **중소벤처24** (smes.go.kr) | REST | 중기부 산하 기관 통합 | 별도 신청 | 공고명, 기간, 지원기관, 첨부파일 |
| **NTIS** (ntis.go.kr) | REST (14개) | 국가 R&D 과제 공고 | 별도 신청 | 과제명, 기간, 예산, 연구분야 |
| **K-Startup** (k-startup.go.kr) | REST | 창업 지원사업 | 별도 신청 | 공고명, 기간, 지원내용 |

### 2.2 크롤링 기반 (2순위 — API 미제공)

| 소스 | 방식 | 커버리지 |
|------|------|---------|
| **보조금24** (gov.kr) | HTML 크롤링 | 개인+기업 보조금 |
| **보조금통합포털** (bojo.go.kr) | HTML 크롤링 | 국고+지방 보조금 |
| **각 부처 공고 페이지** | HTML 크롤링 | 부처별 자체 공모 |

### 2.3 추천 MVP 소스

**Phase 1 (MVP)**: 기업마당 API만으로 시작
- 이유: 가장 포괄적 (중앙+지자체+유관기관 통합), API 안정, 8개 분야 분류 제공
- 공공데이터포털에서 API 키 발급 → 즉시 사용 가능

**Phase 2**: NTIS (R&D) + K-Startup (창업) 추가

**Phase 3**: 보조금24 + 부처별 크롤링 추가

## 3. 기능 설계

### 3.1 데이터 수집 파이프라인

```
기업마당 API 호출 (1일 1회, 09:00 KST)
    ↓
신규 공고 감지 (기존 DB 대비 diff)
    ↓
공고 상세 페이지에서 첨부 PDF 다운로드
    ↓
LLM 요약 생성:
  - 사업명
  - 지원 대상 (업종, 규모, 지역 등)
  - 지원 금액 (최대/최소)
  - 신청 기간 (시작~마감)
  - 신청 방법 (온라인/오프라인/URL)
  - 핵심 자격 요건 3줄
    ↓
SQLite 저장 + 마감일 기준 D-day 계산
    ↓
텔레그램 발송 + 웹사이트 업데이트
```

### 3.2 LLM 요약 포맷 (지원사업 전용)

```
사업명: AI 활용 중소기업 디지털 전환 지원사업
소관: 중소벤처기업부
분야: 기술
지원대상: 종업원 50인 이하 중소기업 (제조업, IT 우선)
지원금액: 기업당 최대 1억원 (자부담 30%)
신청기간: 2026.04.01 ~ 2026.06.30
신청방법: 중소벤처24 온라인 접수
핵심요건:
  1. 중소기업기본법 상 중소기업
  2. AI 도입 계획서 제출
  3. 최근 3년 내 동일 사업 미수혜
```

### 4.2 DB 스키마 (subsidy_cache.db)

```sql
CREATE TABLE subsidies (
    id TEXT PRIMARY KEY,          -- 기업마당 공고 ID
    title TEXT NOT NULL,          -- 사업명
    category TEXT,                -- 분야 (금융/기술/인력/수출/내수/창업/경영/기타)
    ministry TEXT,                -- 소관기관
    executor TEXT,                -- 수행기관
    receiver TEXT,                -- 접수기관
    apply_start TEXT,             -- 신청시작일 (YYYY-MM-DD)
    apply_end TEXT,               -- 신청종료일 (YYYY-MM-DD)
    detail_url TEXT,              -- 상세 URL
    summary TEXT,                 -- AI 요약
    target TEXT,                  -- 지원 대상
    amount TEXT,                  -- 지원 금액
    requirements TEXT,            -- 핵심 자격 요건
    status TEXT DEFAULT 'active', -- active / closed / upcoming
    registered_at TEXT,           -- 등록일
    notified_at TEXT,             -- 알림 발송일
    created_at TEXT DEFAULT (datetime('now','localtime'))
);

CREATE INDEX idx_subsidy_category ON subsidies(category);
CREATE INDEX idx_subsidy_apply_end ON subsidies(apply_end);
CREATE INDEX idx_subsidy_status ON subsidies(status);
```

### 4.3 기업마당 API 호출 예시

```python
import requests

API_KEY = "공공데이터포털_발급키"
BASE_URL = "https://www.bizinfo.go.kr/uss/rss/bizinfoApi.do"

def fetch_subsidies(category="all", page=1):
    params = {
        "crtfcKey": API_KEY,
        "dataType": "json",
        "pageUnit": 100,
        "pageIndex": page,
        "hashtags": "",  # 분야 필터
    }
    r = requests.get(BASE_URL, params=params, timeout=15)
    return r.json()
```

지원사업: "AI 활용 중소기업 디지털 전환 지원" (기업마당 공고)

1. **AI 요약**: 30페이지 PDF를 3줄로 요약 (자격/금액/기간)

3. **법령 연동**: 지원 근거 법령 자동 링크
