## Part 1: 글로벌 영업 전략

### 1.1 타겟 시장 우선순위

| 순위 | 시장 | 진입 시점 | 핵심 근거 |
|:----:|------|---------|----------|
| **1** | **일본** | Year 1 | 한국과 유사한 정부 구조, e-Gov API 존재, 금융청(FSA) RegTech 수요. govbrief 해당 서비스 전무 |
| **2** | **미국** | Year 1-2 | 세계 최대 RegTech 시장 ($12.3B). Federal Register API 정비. 행정부 보도자료 AI 요약 부재 |
| **3** | **싱가포르** | Year 1-2 | 영어권, MAS 핀테크 규제 활발, 동남아 확장 교두보 |
| **4** | **영국** | Year 2 | gov.uk 잘 정비, FCA 규제 모니터링 수요, Brexit 후 규제 추적 부담 |
| **5** | **EU** | Year 2-3 | 27개국 + EU 규제, 다국어 과제 있으나 시장 최대 |

### 2.1 전체 페이지 구성

| # | 페이지 | URL | 핵심 기능 | 우선순위 |

| 1 | **메인 (일일 피드)** | `/` | 히어로 + 카드 그리드 + 사이드 필터 + 검색 | 기존 강화 |

| 5 | **구독/가격** | `/pricing/` | 3단계 가격표 + 기능 비교 + FAQ | **1순위** |
| 6 | **API 문서** | `/api/` | Quick Start + 엔드포인트 + SDK + 코드 예시 | 5순위 |
| 7 | **어바웃** | `/about/` | 미션 + 작동 원리 + 커버리지 맵 + 연락처 | **1순위** |
