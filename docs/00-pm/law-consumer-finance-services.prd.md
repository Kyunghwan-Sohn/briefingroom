## A-1. 내법찾기 (MyLawFinder)

### 한 줄 설명
"나한테 무슨 일이 생겼는데, 어떤 법이 나를 보호하는지" 상황을 말하면 AI가 관련 법조문을 찾아 쉬운 말로 설명해주는 텔레그램 봇

### 구체적 사용 시나리오

> **상황**: 김민수(28세, 직장인)가 전세 만기인데 집주인이 보증금을 안 돌려준다.
>
> 1. 김민수가 텔레그램에서 @MyLawBot에 "전세 보증금 안 돌려줘요. 만기 지났는데 계속 미루고 있어요"라고 입력
> 2. AI가 분석하여 관련 법령 3개를 찾음:
>    - 주택임대차보호법 제3조의2 (보증금의 회수)
>    - 주택임대차보호법 제8조 (보증금 중 일정액의 보호)
>    - 민법 제536조 (동시이행의 항변권)
> 3. 각 조문을 쉬운 말로 해설 + 실제 행동 가이드 제공:
>    - "이 법에 따르면 집주인은 만기일에 보증금을 돌려줘야 합니다"
>    - "임차권등기명령을 신청할 수 있습니다 (비용: 약 5,000원)"
>    - "내용증명 발송 후 소액사건심판 가능 (보증금 3억 이하)"

### 핵심 기능 3가지

1. **상황 기반 법령 검색**: 일상 언어로 상황 설명 -> LLM이 법률 키워드 추출 -> 법제처 API로 관련 법조문 검색 -> 쉬운 해설과 함께 제공
2. **행동 가이드**: 단순히 법조문만 알려주지 않고, "다음에 뭘 해야 하는지" 구체적 절차 안내 (내용증명 보내기, 관할 법원 찾기, 법률구조공단 연락처 등)
3. **유사 판례 연결**: 비슷한 상황의 판례를 1~2건 찾아 "법원이 이런 경우에 어떻게 판단했는지" 요약 제공

### 수익 모델

- **무료**: 일 3회 질문, 기본 법조문 + 쉬운 해설
- **프리미엄**: 월 4,900원 -- 무제한 질문, 판례 연결, 행동 가이드, 서류 양식 안내
- **제휴 광고**: 법률구조공단, 대한법률구조재단, 로톡 등 법률 상담 플랫폼 연결 시 CPA 수수료

### 킬러 포인트: "이거 왜 안 쓸 수가 없지?"

"네이버 지식인에 올리면 3일 걸리고, 답변이 맞는지도 모르겠고, 변호사 상담은 5만원인데... 이건 30초 만에 '법조문 원문 + 쉬운 설명 + 다음 행동'까지 나온다. 게다가 무료다."

### 기존 서비스 대비 차별점

| 기존 서비스 | 한계 | 내법찾기 차별점 |
|------------|------|---------------|
| 네이버 지식인 | 답변 품질 불균일, 법조문 근거 없음, 대기 시간 | 법제처 API 기반 정확한 법조문 + 즉시 응답 |
| 로톡 (AI 진단) | 변호사 연결이 목적, 기본 진단만 무료 | 법조문 직접 제공 + 행동 가이드까지 무료 |
| CaseAI (판례검색) | 판례 중심, 일반인에게 어려움 | 쉬운 말 해설 + 상황 기반 검색 |
| NexusAI법률상담 | 일반적 답변, 법조문 직접 링크 약함 | 법제처 API로 조문 원문 실시간 조회 |
| 국가법령정보센터 | 법률 전문 용어, 자기 상황과 연결 못함 | 일상 언어 -> 법률 매핑을 AI가 수행 |

## A-2. 동네법률 (LocalLegalHelper)

### 한 줄 설명
층간소음, 주차 분쟁, 반려동물, 이웃 갈등 등 생활 분쟁에 대해 "우리 동네 조례까지 포함해서" 해결 방법을 알려주는 서비스

### 구체적 사용 시나리오

> **상황**: 박지영(35세, 아파트 거주)이 윗집 층간소음에 시달린다.
>
> 1. 텔레그램에서 "서울 강남구 아파트인데 윗집 층간소음 너무 심해요. 밤 11시 넘어서도 뛰어다녀요"
> 2. AI가 분석:
>    - 공동주택관리법 제20조 (층간소음의 방지 등)
>    - 소음진동관리법 제21조의2 (층간소음 기준)
>    - 서울특별시 층간소음 분쟁조정에 관한 조례 (지역 특화)
> 3. 행동 가이드:
>    - 1단계: 층간소음 이웃사이센터 (1661-2642) 상담
>    - 2단계: 소음 측정 신청 (무료)
>    - 3단계: 서울시 분쟁조정위원회 조정 신청
>    - 야간(22:00~06:00) 소음 기준: 40dB 초과 시 위반

### 핵심 기능 3가지

1. **지역 맞춤 법률 정보**: 자치법규 API(ordiSearch.do)를 활용하여 사용자 거주 지역의 조례까지 반영한 맞춤 답변
2. **분쟁 유형별 해결 로드맵**: 층간소음/주차/반려동물/공사소음/CCTV/쓰레기 등 15가지 생활 분쟁에 대한 단계별 해결 프로세스 제공
3. **증거 수집 가이드**: 법적 대응 시 필요한 증거(녹음, 사진, 기록지) 작성 방법 및 양식 안내

### 수익 모델

- **무료**: 월 5건 상담, 기본 법조문 + 행동 가이드
- **프리미엄**: 월 3,900원 -- 무제한 상담, 지역 조례 반영, 증거 수집 가이드, 서류 양식
- **제휴**: 이사 서비스, 소음 측정 업체, 법률 상담 플랫폼 연결

### 킬러 포인트

"우리 동네 조례까지 찾아준다. 서울이랑 부산이랑 층간소음 조정 절차가 다른데, 그걸 알아서 구분해준다. 전국 어디서나 내 동네 기준으로 답해준다."

### 기존 서비스 대비 차별점

| 기존 서비스 | 한계 | 동네법률 차별점 |
|------------|------|---------------|
| 층간소음 이웃사이센터 | 전화 상담만, 법조문 설명 없음 | 법조문 + 지역 조례 + 행동 가이드 통합 |
| 정부24 | 민원 접수만, 법적 근거 설명 부족 | 관련 법 + 우리 동네 조례 + 분쟁 해결 로드맵 |
| 네이버 카페 | 개인 경험 위주, 법적 근거 부족 | 법제처 API 기반 정확한 법률 정보 |

## A-3. 계약서 119 (ContractSOS)

### 한 줄 설명
전세/월세 계약서, 근로계약서, 프리랜서 계약서 등을 사진으로 찍으면 "이 조항 주의하세요"를 빨간색/노란색으로 표시해주는 서비스

### 구체적 사용 시나리오

> **상황**: 이수진(25세, 사회초년생)이 첫 원룸 계약을 앞두고 있다.
>
> 1. 부동산에서 받은 계약서를 사진으로 찍어 텔레그램 봇에 전송
> 2. AI가 주요 조항을 분석:
>    - [빨간색] "보증금 반환 시 원상복구 비용을 공제한다" -> "주택임대차보호법 제3조에 따라 통상 마모는 임대인 부담입니다. 이 조항은 불리할 수 있습니다"
>    - [노란색] "계약 해지 시 위약금 보증금의 30%"  -> "민법 제398조 제2항에 따라 부당하게 과다한 위약금은 법원이 감액할 수 있습니다"
>    - [초록색] "임대차기간은 2년으로 한다" -> "주택임대차보호법 제4조에 의한 최소 보장 기간입니다. 적법합니다"
> 3. 체크리스트 제공: "계약 전 확인사항 7가지" (등기부등본 확인, 확정일자 등)

### 핵심 기능 3가지

1. **계약서 촬영 분석**: 텔레그램으로 사진 전송 -> OCR -> LLM 조항 분석 -> 위험도 신호등 표시
2. **법조문 매칭**: 각 조항에 해당하는 민법/주택임대차보호법/근로기준법 등 조문을 자동 연결
3. **계약 유형별 체크리스트**: 전세/월세/근로/프리랜서/매매 등 유형별로 "반드시 확인할 항목" 제공

### 수익 모델

- **무료**: 월 1건 분석, 임대차 계약서 한정
- **프리미엄**: 월 4,900원 -- 월 10건, 전 유형 계약서, 상세 분석
- **제휴**: 부동산 중개 플랫폼(직방, 다방)과 API 연동 수수료

### 킬러 포인트

"변호사한테 계약서 검토 맡기면 10~30만원. 이건 사진 한 장이면 30초 만에 '이 조항 조심하세요'가 나온다. 사회초년생이 전세 사기 피하는 최소 안전장치."

### 기존 서비스 대비 차별점

| 기존 서비스 | 한계 | 계약서119 차별점 |
|------------|------|-----------------|
| 로톡 계약서 검토 | 건당 10~30만원, 변호사 대기 | 무료/저가, 즉시 분석 |
| 전세사기 예방 앱(HUG) | 보증보험 가입 여부만 확인 | 계약서 조항 자체를 분석 |
| 직방/다방 계약 가이드 | 일반적 주의사항만 제공 | 내 계약서의 구체적 조항을 법조문과 대조 |

## A-4. 퇴직금 계산기+ (MyExitPay)

### 한 줄 설명
"나 퇴직금 받을 수 있어?" 근무 조건을 입력하면 퇴직금 계산 + 관련 법조문 + 미지급 시 대응법까지 알려주는 노동법 특화 서비스

### 구체적 사용 시나리오

> **상황**: 최영호(32세, 편의점 알바 3년차)가 사장이 퇴직금을 안 준다.
>
> 1. 텔레그램 봇에 근무 조건 입력: 주 30시간, 3년 근무, 월 200만원
> 2. AI 분석:
>    - 근로기준법 제34조: "사용자는 계속근로기간 1년에 대하여 30일분 이상의 평균임금을 퇴직금으로 지급하여야 한다"
>    - 계산: 퇴직금 약 600만원 (200만원 x 3년)
>    - 근로기준법 제36조: "퇴직 후 14일 이내 지급 의무"
> 3. 미지급 대응 안내:
>    - 1단계: 노동부 고용노동부 진정 (온라인: minwon.moel.go.kr)
>    - 2단계: 체불 임금 지급명령 신청
>    - 3단계: 소액사건심판 (소송비용 약 5,000원)
>    - 근로기준법 제109조: 미지급 시 3년 이하 징역 또는 3천만원 이하 벌금

### 핵심 기능 3가지

1. **퇴직금/임금 자동 계산**: 근무 조건(기간, 급여, 근무시간) 입력 -> 법정 퇴직금 및 미지급 수당(연차, 야근 등) 계산
2. **법조문 근거 제시**: 계산 결과마다 근거가 되는 근로기준법/최저임금법 조문을 쉬운 말로 함께 표시
3. **미지급 대응 프로세스**: 사업주가 안 줄 때의 단계별 대응법 (진정 -> 지급명령 -> 소송) + 각 단계별 필요 서류 안내

### 수익 모델

- **무료**: 기본 퇴직금 계산 + 법조문 1건
- **프리미엄**: 월 3,900원 -- 연차/야근/주휴 수당 정밀 계산, 진정서 양식 생성, 판례 연결
- **B2G 제휴**: 고용노동부/근로복지공단 공식 연계

### 킬러 포인트

"퇴직금 계산기는 많지만, 법적 근거 + 안 주면 어떻게 하는지까지 알려주는 건 없다. 사장이 '너 퇴직금 대상 아니야'라고 하면, 이 봇이 법조문으로 반박해준다."

### 기존 서비스 대비 차별점

| 기존 서비스 | 한계 | 퇴직금계산기+ 차별점 |
|------------|------|-------------------|
| 고용노동부 퇴직금 계산기 | 단순 계산만, 법적 대응 없음 | 계산 + 법조문 + 대응 절차 통합 |
| 알바몬/잡코리아 계산기 | 퇴직금만 계산, 부당 해고/임금 체불 무관 | 노동법 전반 커버 (해고, 수당, 산재 포함) |

## A-5. 쇼핑법률 도우미 (ShopRights)

### 구체적 사용 시나리오

> **상황**: 정하나(29세)가 인스타 마켓에서 산 옷이 사진과 전혀 달라서 환불 요청했는데 판매자가 거부한다.
>
> 1. 텔레그램 봇에 상황 설명: "인스타에서 옷 샀는데 색상이 완전 달라요. 환불 요청했더니 주문제작이라 환불 불가라고 해요. 받은 지 3일 됐어요"
> 2. AI 분석:
>    - 전자상거래법 제17조 제1항: "소비자는 재화를 공급받은 날로부터 7일 이내에 청약철회 가능"
>    - 전자상거래법 제17조 제3항: "표시/광고 내용과 다른 경우 30일 이내 청약철회 가능"

> 3. 대응 가이드:
>    - "판매자에게 이 법조문을 보여주면서 환불 요청하세요"
>    - "거부 시: 1372 소비자상담센터 -> 한국소비자원 피해구제 신청"
>    - 판매자에게 보낼 메시지 템플릿 제공

1. **소비자 분쟁 유형 진단**: 환불 거부/하자 상품/허위 광고/배송 지연/개인정보 유출 등 유형별 관련 법조문 즉시 제공
2. **판매자에게 보낼 메시지 생성**: 법적 근거가 포함된 환불 요청/항의 메시지를 AI가 작성해주는 기능
3. **소비자 구제 절차 안내**: 1372 -> 소비자원 -> 소액소송 단계별 가이드 + 각 단계 소요 기간/비용 안내

### 수익 모델

- **무료**: 월 5건 상담, 기본 법조문 제공
- **프리미엄**: 월 3,900원 -- 무제한, 메시지 생성, 내용증명 양식
- **제휴**: 소비자 피해 법률 상담 연결 CPA

## A-6. 가족법 가이드 (FamilyLawGuide)

### 한 줄 설명
이혼, 상속, 양육권, 재산 분할 등 가족법 문제에서 "내 권리는 뭐고, 절차는 어떻게 되는지" 단계별로 안내하는 서비스

### 구체적 사용 시나리오

> **상황**: 김수미(45세)의 어머니가 돌아가시고, 형제 3명이 상속 문제로 갈등이 있다.
>
> 1. "어머니가 돌아가셨는데 유언장이 없어요. 형제 3명이고 아버지는 생존. 아파트 1채랑 예금 있어요"
> 2. AI 분석:
>    - 민법 제1000조 (상속 순위): "배우자 + 직계비속이 공동상속"
>    - 민법 제1009조 (법정상속분): "배우자 1.5 : 자녀 각 1" -> 아버지 1.5, 형제 각 1 (총 4.5 기준)
>    - 민법 제1019조: "상속 개시를 안 날로부터 3개월 내 승인/포기"
> 3. 계산 + 절차:
>    - 아버지 상속분: 전체의 1.5/4.5 = 33.3%
>    - 각 자녀 상속분: 전체의 1/4.5 = 22.2%
>    - 다음 절차: 사망신고(1개월 내) -> 상속재산 조회(안심상속) -> 상속세 신고(6개월 내)

### 핵심 기능 3가지

1. **상속분 계산기**: 가족 관계 입력 -> 법정 상속분 자동 계산 (배우자, 직계비속/존속, 형제자매 등 경우의 수 처리)
2. **이혼 절차 가이드**: 협의이혼/재판이혼 판단 + 양육권/재산분할/위자료 관련 법조문 + 절차 타임라인
3. **상속/이혼 판례 연결**: 유사 상황의 판례를 찾아 "법원이 어떻게 판단했는지" 참고 제공

### 수익 모델

- **무료**: 기본 상속분 계산 + 법조문 조회
- **프리미엄**: 월 4,900원 -- 상세 절차 가이드, 판례 연결, 서류 양식
- **제휴**: 상속 전문 법무사/세무사 연결 수수료

### 킬러 포인트

"상속 문제가 터지면 형제 사이가 깨진다. 법적으로 각자 얼마를 받을 수 있는지 객관적 숫자를 보여주면, 감정 싸움 대신 팩트 기반 대화가 가능해진다."

### 기존 서비스 대비 차별점

| 기존 서비스 | 한계 | 가족법가이드 차별점 |
|------------|------|------------------|
| 법원 상속 안내 | 일반적 설명, 개인화 계산 없음 | 내 가족 관계 기반 정확한 계산 |
| 상속세 계산기 (국세청) | 세금만, 분할/절차 무관 | 상속분 + 절차 + 세금 통합 |
| 이혼 전문 로펌 상담 | 건당 10만원+, 심리적 장벽 | 무료/저가, 비대면, 법조문 근거 제공 |

## A-7. 교통사고 법률 119 (AccidentLaw119)

### 한 줄 설명

>    - [긴급 행동] 사진 촬영 -> 보험사 연락 -> 경찰 신고 (도로교통법 제54조)
>    - [과실 참고] 직진 vs 차선변경 사고 참고 과실비율: 상대 80~90% : 나 10~20%
>    - [관련 법률] 도로교통법 제19조 (안전거리확보), 교통사고처리특례법 제3조

2. **과실 비율 참고**: 사고 유형(직진 vs 좌회전, 후진 사고 등) 별 일반적 과실 비율 안내 + 근거 판례
3. **보험/합의 가이드**: 보험 처리 절차, 합의금 산정 참고, 자동차손해배상보장법 관련 조문

### 수익 모델

- **무료**: 긴급 체크리스트 + 기본 법조문
- **프리미엄**: 월 3,900원 -- 과실 비율 상세 분석, 합의 가이드, 판례 검색
- **제휴**: 자동차 보험사, 차량 수리 업체 연결

### 킬러 포인트

"사고 나면 머리가 하얘진다. 그 순간 이 봇 켜면 '지금 사진 찍으세요 -> 보험사 전화하세요 -> 이런 사고는 보통 상대 과실이 더 높습니다'가 바로 나온다."

### 기존 서비스 대비 차별점

|------------|------|---------------------|

### DB 스키마

```sql
-- 사용자 프로필
CREATE TABLE fin_users (
    user_id TEXT PRIMARY KEY,        -- 텔레그램 user_id
    company_type TEXT,               -- bank/securities/insurance/card
    tier TEXT DEFAULT 'starter',     -- starter/pro/enterprise
    created_at TEXT
);

-- 금융 법령 워치리스트
CREATE TABLE fin_watchlist (
    user_id TEXT,
    law_name TEXT,
    law_category TEXT,               -- common/bank/securities/insurance/card
    keywords TEXT,                   -- JSON array
    created_at TEXT,
    PRIMARY KEY(user_id, law_name),
    FOREIGN KEY(user_id) REFERENCES fin_users(user_id)
);

-- 법령 스냅샷 (변경 감지용)
CREATE TABLE fin_law_snapshot (
    law_name TEXT PRIMARY KEY,
    law_mst TEXT,
    enforcement_date TEXT,
    promulgation_date TEXT,
    articles_hash TEXT,
    full_text TEXT,
    snapshot_at TEXT
);

-- 변경 이력
CREATE TABLE fin_law_changes (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    law_name TEXT,
    change_type TEXT,                -- amendment/new/abolish
    old_enforcement TEXT,
    new_enforcement TEXT,
    changed_articles TEXT,           -- JSON: 변경된 조문 목록
    diff_text TEXT,                  -- diff 결과
    ai_summary TEXT,                 -- AI 영향 분석
    related_sanctions TEXT,          -- JSON: 관련 제재 사례
    detected_at TEXT
);

-- 금감원 제재 사례
CREATE TABLE fin_sanctions (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    company_name TEXT,
    company_type TEXT,
    violation_law TEXT,              -- 위반 법조문
    sanction_type TEXT,              -- fine/warning/suspension
    sanction_amount INTEGER,         -- 과태료/과징금 금액
    sanction_date TEXT,
    summary TEXT,
    source_url TEXT
);

-- 알림 발송 기록
CREATE TABLE fin_notifications (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    user_id TEXT,
    change_id INTEGER,
    sent_at TEXT,
    read_at TEXT,
    FOREIGN KEY(change_id) REFERENCES fin_law_changes(id)
);
```

# 부록: 법적 고려사항

## 변호사법 제109조 관련

현행 변호사법은 비변호사의 법률사무 취급을 금지합니다. 본 서비스는 다음과 같이 법률 정보 제공과 법률 자문을 명확히 구분합니다:

| 구분 | 법률 정보 제공 (허용) | 법률 자문 (금지) |
|------|---------------------|----------------|
| 정의 | 법조문/판례 원문 제공 + 일반적 해설 | 구체적 사안에 대한 법적 판단/조언 |
| 본 서비스 | 법제처 API 원문 + AI 일반 해설 | 하지 않음 |
| 면책 문구 | 모든 응답에 면책 조항 포함 | - |
| 연결 서비스 | 법률구조공단/변호사 상담 안내 | - |

면책 조항 예시:
> "이 정보는 법제처 Open API에서 제공하는 법률 정보를 AI가 정리한 것입니다.
> 법률 자문이 아닌 일반적인 법률 정보이며, 구체적인 법적 판단이 필요한 경우
> 변호사와 상담하시기 바랍니다."
