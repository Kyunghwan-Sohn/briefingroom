1홈 진입 (govbrief.kr)
사용자가 처음 방문하는 화면. 검색 + 2분할 캐러셀 + 정책요약 + 부처별 브리핑
홈 govbrief.kr/
헤더: 브리핑룸 | 홈(활성) | 정부정책AI | 금융법령AI | 아카이브 | 알림
검토: 검색 → Supabase ILIKE 폴백 동작 확인. 2분할 캐러셀 정상 여부. 부처 탭 필터링 JS 동작 여부.

2정부 정책 AI (govbrief.kr/policy/)
홈에서 "정부 정책 AI" 탭 클릭 또는 캐러셀 "전체 보기" 클릭
정부 정책 AI govbrief.kr/policy/
히어로: 파랑 배경 + 숫자 4개 (발표건수/참여기관/영향도상/스냅샷) + 검색
검토: 분야 탭(금융경제/산업기술/...) 필터링 동작. 카드 클릭 시 상세 페이지 이동. 부처별 탭 필터.

3보도자료 상세 페이지

배지(기관명) 16px / 메타(날짜/분야/기관) 16px / 요약 + 키워드 + 원문링크
검토: 분야 필터(금융경제 등) 클릭 시 같은 날짜 필터. 줄간격 1.55 적용 여부. 상세 브리핑(왜 중요한가 등) 존재 여부.
