규제 구조도

/finlaw/map/

주제 또는 조문을 선택하면 법 → 시행령 → 감독규정 → 시행세칙 순으로 관련된 규정 체인을 위에서 아래로 펼쳐 보여줌. 하나의 규제가 어떤 상위법에 근거하고 어떤 하위 규정으로 구체화되는지 한번에 파악.

  • 주제/조문 검색 → 해당 체인 자동 구성
  • 법 · 시행령 · 감독규정 · 시행세칙 단계별 표시
  • 각 단계에서 해당 조문만 추출 인용
  • 위임 관계 (법 §X → 시행령 §Y) 명시
  • 특정 조문 클릭 → 개별 법령 상세로 이동
  • 각 층에서 최근 개정 여부 배지 표시
