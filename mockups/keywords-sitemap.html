트리맵 메인

/keywords/

키워드 분석 섹션의 메인. 최근 30일 (또는 선택 기간) 정부 보도자료에서 빈도 기반으로 추출한 키워드를 트리맵으로 시각화. 크기 = 언급 빈도, 색 = 카테고리.

  • 기간 선택 (7일 / 30일 / 90일 / 커스텀)
  • 트리맵 (카테고리별 색상, 빈도별 크기)
  • 카테고리 범례
  • Top 10 키워드 랭킹 (전주 대비 등락)
  • 각 셀 클릭 → 개별 키워드 드릴다운

개별 키워드 드릴다운

/keywords/{slug}/

트리맵에서 키워드 클릭 시 진입. 해당 키워드가 포함된 보도자료 전체, 언급한 부처 분포, 30일 시계열 추이를 보여줌.

  • 키워드 헤더 (카테고리, 총 언급 수, 전주 대비)
  • 시계열 미니 차트 (일별 언급 수)
  • 부처별 언급 블록 — 각 부처 내 발표 리스트
  • 연관 키워드 (공출현 빈도 기반)
  • "이 키워드를 AI에 질문" CTA → /brief/ai/?q={keyword}

산업 반응 분석

/keywords/industry/

정부 키워드에 대해 국내 뉴스가 어떻게 반응하는지 분석. "정부가 X를 발표했을 때 산업·시장은 실제로 어디에 집중하는가"를 보여줌. 정부 의도와 시장 현안 사이의 갭을 드러내는 시장 차별화 기능.

  • 키워드 선택 → 관련 뉴스 기사 집계
  • 반응 큰 산업 Top 5 (뉴스 언급 기업·섹터 기반)
  • 산업 관점 주요 우려 (뉴스 헤드라인 클러스터링)
  • 정부 방향 vs 산업 현안 갭 분석
  • 시계열 — 정부 발표 후 뉴스 반응 지연·증폭 곡선

카테고리 뷰

/keywords/categories/

카테고리별로 상위 키워드를 묶어 보여줌. 금융·경제 / 산업·기술 / 사회·복지 / 외교·안보 / 행정·법제 5개 축으로 "어느 영역에서 무슨 일이 벌어지는지" 파악.

  • 카테고리 카드 그리드 (5개)
  • 각 카드 — 카테고리 내 Top 10 키워드
  • 카테고리별 발표 건수·참여 기관
  • 카테고리 선택 시 해당 카테고리 트리맵으로 이동
